Text Compression

For my text compression project, I developed a Huffman Encoding Tree in Python, which efficiently compresses text files by assigning optimized bit encodings based on character frequency. The program reads from a file, analyzes the frequency of each character, and constructs a binary tree ( Huffman Tree) where more frequent characters are assigned shorter binary codes. This approach significantly reduces the size of the file while maintaining all the original data, ensuring lossless compression. The project demonstrates my ability to implement algorithms for practical applications in data compression and highlights my proficiency in Python and problem-solving with efficient coding techniques.
